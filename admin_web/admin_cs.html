🟡 카카오톡 비즈니스 채널 연동 (CS 방어막)

앱 내에서 '카카오톡 상담' 버튼 클릭 시 연결될 채널 정보와 동작 방식을 설정합니다.
💡 운영 팁: 1:1 채팅은 최소화하고, 카카오톡 챗봇(스마트채팅) 기능을 활성화하여 단순 문의를 1차적으로 걸러내는 것을 강력히 권장합니다.
카카오 비즈니스 채널 ID (검색용)
운영 중인 카카오톡 채널의 고유 검색용 아이디를 입력하세요. (예: @pickle_cs)
카카오톡 채널 다이렉트 링크 (URL)
앱 내 버튼에 연동될 실제 채널 주소입니다.
1:1 상담원 채팅 연결 허용 시간 설정
해당 시간 외에는 유저가 1:1 채팅을 걸 수 없도록 시스템 안내 메시지를 송출합니다.
